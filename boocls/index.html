bucles

1. Bucle for

Este bucle es como un contador. Te permite repetir un bloque de código un número específico de veces.

for (let i = 1; i <= 5; i++) { console.log(i); }

2. Bucle while

Este bucle repite un bloque de código mientras una condición sea verdadera

let numero = 0; while (numero <= 10) { numero = parseInt(prompt("Introduce un número mayor que 10:")); } console.log("¡Número válido!");
